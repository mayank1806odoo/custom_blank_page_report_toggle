Odd Page Blank Report

This module automatically inserts a blank page after every odd-numbered page in both standard Odoo base reports and custom reports. It ensures your documents have a professional layout, ideal for double-sided printing.



Powerful Features at a Glance

  • Blank Page After Odd Pages: Inserts a clean blank page after every odd-numbered page in your report.
  • Universal Support: Works seamlessly with both core Odoo reports and custom-built ones.
  • Odoo 18 Ready: Designed and tested for complete compatibility with Odoo 18's new engine.
  • Zero Code Setup: Just enable a checkbox in report settingsâ€”no developer help needed.
  • Double-Sided Perfection: Optimizes print layout for professional double-sided documents.


Configuration Instructions

Step 1: Navigate to Settings > Technical > Reports > Reports.

Step 2: Select the report you wish to modify from the list.

Step 3: Scroll to the Advanced Properties section and enable the checkbox labeled Enable Blank Page Padding.

This will automatically insert a blank page after each odd-numbered page, perfect for clean and professional double-sided printing.
